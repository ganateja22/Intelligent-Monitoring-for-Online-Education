# Intelligent-Monitoring-for-Online-Education
AI-powered student attention tracking system using computer vision and machine learning to monitor engagement in real time. It uses face detection, eye tracking, and attention scoring to analyze focus, achieving 85.7% accuracy. Provides alerts and analytics to improve online learning outcomes.

## 🚀 Quick Start

### Option 1: Easy Installation & Run
1. **Double-click `install.bat`** - Installs all required packages
2. **Double-click `run_simple.bat`** - Starts basic attention tracking
3. **Double-click `run_enhanced.bat`** - Starts full tracking with analytics

### Option 2: Manual Installation
```bash
pip install -r requirements.txt
python simple_demo.py          # Basic version
python enhanced_tracker.py     # Full version with analytics
python attention_analytics.py  # Generate graphs only
```
